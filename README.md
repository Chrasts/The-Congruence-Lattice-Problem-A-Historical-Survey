The Congruence Lattice Problem: A Historical Survey

This repository contains a short historical survey of the Congruence Lattice Problem (CLP) in lattice theory.

The text focuses on the main milestones and ideas in the development of the problem, with only brief sketches of proofs. The goal is to present the historical development and key results in a concise and accessible form.

Files

clp_survey.tex — LaTeX source of the article

clp_survey.pdf — compiled PDF version

Build

To compile the document from the source file:

latexmk -pdf paper.tex

or with standard LaTeX tools:

pdflatex paper.tex
